# Quick Start Scripts

This directory contains helper scripts to quickly run the application.

## Windows (PowerShell)

### Start Backend
```powershell
.\start_backend.ps1
```

### Start Frontend
```powershell
.\start_frontend.ps1
```

## Linux/Mac (Bash)

### Start Backend
```bash
./start_backend.sh
```

### Start Frontend
```bash
./start_frontend.sh
```
